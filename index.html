Project Highlights

HolonomyJuly 2022

Made in 48 hours for the GMTK Game Jam, my platformer achieved 2nd place among over 6,000 competing game jam entries. This game demonstrates novel graphics techniques, allowing the 2D player character to navigate on the surface of a 3D world. Custom physics logic allows the 2D physics of the player to interact seamlessly with the geometry of the 3D environment. The game features a striking visual style, implemented via custom material and post- processing shaders.

FuzzpaintOctober 2021 - Current

A work-in-progress vector graphics software aimed at digital artists, written in Rust (formerly C++) and featuring graphics acceleration with the Vulkan API. The aim is to make working with vector graphics as intuitive as working with raster images, but allowing for editing capabilities and visual effects impossible in a traditional layered image editing workflow â€“ for example, retroactively modifying brush settings after a drawing is complete. Over several iterations, I have implemented features including dynamic tesselation of vector objects in a compute shader, signed distance field effects, and efficient rendering of mixed objects of differing shader materials.

NESRemixAugust 2023 - Current

A (yet unreleased) NES emulator providing the user the ability to create and share custom graphics packs. Planned features include allowing the user to add particle effects, lighting, raymarched shadows, PBR textures, screenspace effects, etc. to their favorite vintage games. This is an exciting deep-dive into pixel-perfect rendering as well as hardware emulation, and I can't wait to show it off!
